AP Chemistry Game

Context

For our AP Chemistry project, I created an interactive game designed to help students practice and memorize the common charges of chemical elements. The goal was to turn a challenging memorization task into a fun, engaging learning experience.The project involved planning out question logic, designing the interface, and coding with HTML, CSS, and JavaScript. I wanted the game to be clear, quick to play, and provide helpful feedback for learners.

Content

The game includes the following features:

  • A quiz-style format with open-ended questions about element charges
  • Different levels
  • Score Display
  • Randomized questions each round to keep it fresh

I used:

  • HTML to structure the game elements
  • CSS for layout, color themes, and feedback animations
  • JavaScript for logic, question randomization, and interactivity

Challenges

One major challenge was balancing the difficulty, some elements have multiple charges, and I had to decide how to present that clearly to users. I solved this by focusing on the most common or AP-relevant charges.

Takeaways

  • Making learning interactive helps with memory retention
  • JavaScript is powerful for building educational tools
  • Planning logic before coding saves time and confusion later

Next Steps

In the future, I’d like to expand the game to include polyatomic ions, timed challenges, and progress tracking to support long-term studying.
