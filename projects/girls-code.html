Girls Code Project

Context

This project is a website I created to raise awareness and share resources for girls interested in learning how to code. My goal was to inspire more young women to explore careers in technology by showing them that coding is not only possible for them, but also exciting and empowering. The site addresses the gender gap in STEM by providing a safe, encouraging space where girls can access information, guidance, and opportunities to grow their skills.

Content

The website is divided into several key sections:

  • Why Coding? Explains the importance of coding in today’s world and the opportunities it can open.
  • Learning Resources: Links to free and beginner-friendly coding courses, tutorials, and platforms, organized by age group and skill level.
  • Inspiring Stories: Profiles of women in tech who share their journeys and advice.

Challenges

One of the biggest challenges was narrowing down the vast number of resources into a collection that would be helpful and not overwhelming. I also needed to make sure the content was clear for beginners while still offering value for those with more experience. Another challenge was creating a visually appealing layout that still loaded quickly and worked well on different devices.

Takeaways

I learned how to structure a website with clear navigation, present information in a way that motivates users, and use design choices to make a positive impact. I also deepened my understanding of the importance of representation in STEM.

Next Steps

In future projects, I want to add interactive features like a mentorship sign-up, coding challenges, and a forum for girls to share their projects and ask questions.
